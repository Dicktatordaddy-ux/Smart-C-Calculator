# Smart C Calculator

A menu-driven calculator program written in C that performs common mathematical operations with proper error handling and modular function design.

## Features
- Addition
- Subtraction
- Multiplication
- Division with divide-by-zero protection
- Power calculation
- Square root with negative number handling
- Continuous operation until user exits

## How to Run

1. Copy the file `smart_calculator.c`
2. Compile using:
   gcc smart_calculator.c -o calculator -lm
3. Run:
   ./calculator
